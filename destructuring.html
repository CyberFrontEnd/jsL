Разрушения массива в javascript

В JavaScript под разрушением (destructuring) массива понимается удобный способ извлечения значений из массивов и объектов и присваивания их переменным. Это позволяет существенно упростить и улучшить читаемость кода. Давайте рассмотрим несколько примеров разрушения массивов.

Основное разрушение массива

const array = [1, 2, 3, 4, 5]; // Извлекаем первые два элемента const [first, second] = array; console.log(first); // 1 console.log(second); // 2

Пропуск элементов

Вы можете пропускать элементы массива, используя запятые:

const array = [1, 2, 3, 4, 5]; // Извлекаем первый и третий элемент, пропуская второй const [first, , third] = array; console.log(first); // 1 console.log(third); // 3

Остаточные элементы

Можно использовать оператор ... для сбора оставшихся элементов в отдельный массив:

const array = [1, 2, 3, 4, 5]; // Извлекаем первые два элемента, остальные собираем в отдельный массив const [first, second, ...rest] = array; console.log(first); // 1 console.log(second); // 2 console.log(rest); // [3, 4, 5]

Значения по умолчанию

Вы можете задавать значения по умолчанию для переменных:

const array = [1, 2]; // Извлекаем первые три элемента, задавая значение по умолчанию для третьего const [first, second, third = 3] = array; console.log(first); // 1 console.log(second); // 2 console.log(third); // 3

Вложенное разрушение

Массивы могут содержать вложенные массивы, и их тоже можно разрушать:

const array = [1, [2, 3], 4]; // Извлекаем элементы, включая вложенные const [first, [second, third], fourth] = array; console.log(first); // 1 console.log(second); // 2 console.log(third); // 3 console.log(fourth); // 4

Обмен значений переменных

С помощью разрушения можно легко менять местами значения двух переменных:

let a = 1; let b = 2; // Меняем местами значения переменных [a, b] = [b, a]; console.log(a); // 2 console.log(b); // 1

Разрушение массивов — это мощный инструмент, который позволяет писать более чистый и понятный код, особенно при работе с массивами и объектами.
