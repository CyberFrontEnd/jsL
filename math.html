Объект Math

  1. Math.abs(x): возвращает абсолютное значение числа.

    Math.abs(-10); // returns 10

  2. Математический round(x): возвращает значение числа, округленное до ближайшего целого числа.

    Math.round(4.7); // returns 5

  3. Math.ceil(x): возвращает наименьшее целое число, большее или равное числу.

    Math.ceil(4.3); // returns 5

  4. Math.floor(x): возвращает наибольшее целое число, меньшее или равное числу.

    Math.floor(4.7); // returns 4

  5. Math.max(x1, x2, ..., xn): возвращает наибольшее из нуля или более чисел.

    Math.max(10, 20, 30); // returns 30

  6. Math.min(x1, x2, ..., xn): возвращает наименьшее из нулевых или более чисел.

    Math.min(10, 20, 30); // returns 10

  7. Math.random(): Возвращает случайное число с плавающей запятой в диапазоне от 0 (включительно) до 1 (без учета).

    Math.random(); // returns a random number between 0 and 1

  8. Math.pow(x, y): возвращает основание в степень экспоненты, то есть x ^y.

    Math.pow(2, 3); // returns 8

  9. Math.sqrt(x): возвращает квадратный корень из числа.

    Math.sqrt(16); // returns 4

  10. Math.sin(x), Math.cos(x), Math.tan(x): возвращает синус, косинус и тангенс числа (заданного в радианах) соответственно.

    Math.sin(Math.PI / 2); // returns 1 (sine of 90 degrees)

Это всего лишь несколько примеров методов, доступных в Math объекте в JavaScript. Существует больше методов для выполнения других математических операций, тригонометрических функций, логарифмических функций и т.д.
