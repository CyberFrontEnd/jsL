что делает метод filter в js

В JavaScript метод filter() используется для фильтрации элементов массива на основе определённого условия и создания нового массива, содержащего только те элементы, для которых условие истинно. Этот метод не изменяет исходный массив, а возвращает новый массив с отфильтрованными элементами.

Синтаксис метода filter() выглядит так:
let newArray = array.filter(callback(element[, index[, array]])[, thisArg])

function(currentValue, index, array): Это функция, которая будет применяться к каждому элементу массива. Она может принимать до трех параметров:

  1. callback - это функция, которая будет вызвана для каждого элемента массива. Она должна возвращать true, чтобы элемент остался в новом массиве, или false, чтобы элемент был исключён.

  2. element - текущий обрабатываемый элемент массива.

  3. array (необязательный) - сам массив, который был пройден.

  4. thisArg (необязательный) - значение, используемое как this при вызове callback.

let numbers = [1, 2, 3, 4, 5, 6]; let evenNumbers = numbers.filter(function(num) { return num % 2 === 0; }); console.log(evenNumbers); // Output: [2, 4, 6]

В этом примере массив evenNumbers будет содержать только чётные числа из массива numbers, потому что функция обратного вызова возвращает true только для чётных чисел.

примеры метода filter() в js

  1. Фильтрация массива чисел для получения только четных чисел:

    const numbers = [1, 2, 3, 4, 5, 6]; const evenNumbers = numbers.filter(num => num % 2 === 0); console.log(evenNumbers); // Output: [2, 4, 6]

  2. Фильтрация массива строк для получения только тех, которые начинаются с определенной буквы:

    const fruits = ['apple', 'banana', 'orange', 'kiwi', 'grape']; const fruitsStartingWithA = fruits.filter(fruit => fruit.startsWith('a')); console.log(fruitsStartingWithA); // Output: ['apple']

  3. Фильтрация массива объектов на основе свойства:

    const people = [ { name: 'John', age: 30 }, { name: 'Alice', age: 25 }, { name: 'Bob', age: 35 }, { name: 'Eve', age: 40 } ]; const peopleOverThirty = people.filter(person => person.age > 30); console.log(peopleOverThirty); // Output: [{ name: 'Bob', age: 35 }, { name: 'Eve', age: 40 }]

  4. Отфильтровывание нулевых или неопределенных значений из массива:

    const mixedArray = [1, null, 'apple', undefined, 5, 'banana']; const filteredArray = mixedArray.filter(item => item !== null && item !== undefined); console.log(filteredArray); // Output: [1, 'apple', 5, 'banana']

  5. Использование аргумента index для фильтрации повторяющихся элементов:

    const numbers = [1, 2, 3, 2, 4, 5, 4, 6]; const uniqueNumbers = numbers.filter((num, index, array) => array.indexOf(num) === index); console.log(uniqueNumbers); // Output: [1, 2, 3, 4, 5, 6]

Эти примеры демонстрируют различные варианты использования filter() метода для извлечения определенных элементов из массивов на основе различных условий.

в чем разница между методами .forEach() и .map() в js

В JavaScript оба .forEach() и .map() являются методами, используемыми для перебора массивов. Однако они служат несколько иным целям:

  1. forEach():

    Назначение: forEach() Метод выполняет итерацию по каждому элементу в массиве и позволяет вам выполнить указанное действие для каждого элемента.

    Поведение: он не создает новый массив; вместо этого он выполняет предоставленную функцию один раз для каждого элемента массива.

    Использование: Обычно используется, когда вы хотите выполнить определенную операцию над каждым элементом массива без изменения самого массива.

    const numbers = [1, 2, 3, 4]; numbers.forEach(number => console.log(number * 2)); // Output: 2, 4, 6, 8

  2. map():

    Назначение: map() Метод создает новый массив путем применения функции к каждому элементу в исходном массиве и сбора результатов.

    Поведение: возвращает новый массив с результатами вызова предоставленной функции для каждого элемента в вызывающем массиве.

    Использование: Обычно используется, когда вы хотите преобразовать каждый элемент массива определенным образом и собрать результаты в новый массив.

    const numbers = [1, 2, 3, 4]; const doubledNumbers = numbers.map(number => number * 2); console.log(doubledNumbers); // Output: [2, 4, 6, 8]

  3. filter():

    Назначение: filter() Метод создает новый массив со всеми элементами, которые проходят тест, реализованный provide

    Поведение: возвращает новый массив с элементами, удовлетворяющими условию, указанному в функции обратного вызова.

    Назначение: filter() Метод создает новый массив со всеми элементами, которые проходят тест, реализованный provide

    const numbers = [1, 2, 3, 4]; const evenNumbers = numbers.filter(number => number % 2 === 0); console.log(evenNumbers); // Output: [2, 4]

Вкратце, forEach() предназначен для выполнения функции над каждым элементом массива без создания нового массива, map() предназначен для преобразования каждого элемента массива и сбораfilter() предназначен для создания нового массива, содержащего только элементы, удовлетворяющие указанному условию.
