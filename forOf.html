Оператор for of

for...of — это цикл в JavaScript, который позволяет перебирать итерируемые объекты (такие, как массивы, строки, объекты Map, Set и т.д.). Он был добавлен в ECMAScript 6 (ES6) и является более современным и удобным способом итерирования по элементам коллекций по сравнению с другими циклами, такими как for и for...in.

Синтаксис

for (const element of iterable) { // код, который будет выполняться для каждого элемента }
  1. element — переменная, которая будет содержать текущий элемент итерируемого объекта на каждой итерации.
  2. iterable — итерируемый объект, по которому происходит проход (например, массив, строка и т.д.).

Перебор массива

const array = [1, 2, 3, 4, 5]; for (const number of array) { console.log(number); } // Вывод: // 1 // 2 // 3 // 4 // 5

Перебор строки

const string = "Hello"; for (const char of string) { console.log(char); } // Вывод: // H // e // l // l // o

Перебор объекта Map

const map = new Map(); map.set('name', 'John'); map.set('age', 30); for (const [key, value] of map) { console.log(`${key}: ${value}`); } // Вывод: // name: John // age: 30

Перебор объекта Set

const set = new Set([1, 2, 3, 4, 5]); for (const value of set) { console.log(value); } // Вывод: // 1 // 2 // 3 // 4 // 5

Отличия от for...in

for...of предназначен для перебора значений итерируемого объекта.

for...in предназначен для перебора свойств объекта.

const array = [1, 2, 3, 4, 5]; for (const index in array) { console.log(index); } // Вывод: // 0 // 1 // 2 // 3 // 4

Как видно, for...in перебирает индексы (или ключи), тогда как for...of перебирает значения.

Цикл for...of является мощным инструментом для итерирования по коллекциям в JavaScript и делает код более читаемым и лаконичным.
