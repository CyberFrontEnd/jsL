В JavaScript .contains метод используется в разных контекстах в зависимости от типа объекта, к которому он вызывается. Наиболее распространенные варианты использования - с Node интерфейсом и DOMTokenList interface.

  1. Node.contains()

    Метод Node.contains() проверяет, является ли узел потомком данного узла. Он возвращает, true если данный узел является потомком узла, на котором он вызывается, включая сам узел, и false в противном случае.

    let parent = document.getElementById('parent'); let child = document.getElementById('child'); console.log(parent.contains(child)); // истинно, если "дочерний" является потомком "родительского"

  2. DOMTokenList.contains()

    Метод DOMTokenList.contains() проверяет, присутствует ли данный класс (или токен) в списке токенов (обычно это имена классов в элементе). Он возвращает, true если класс присутствует, и false в противном случае.

    let element = document.getElementById('myElement'); console.log(element.classList.contains('myClass')); // истинно, если "мой класс" является одним из классов в "моем элементе"

    1. Подробные сведения:

      Node.contains() часто используется для проверки, находится ли определенный элемент внутри другого элемента в дереве DOM. Это может быть полезно для обработки событий и манипулирования DOM.

    2. DOMTokenList.contains()

      в основном используется для проверки наличия класса в элементе. Это может быть полезно для условного применения стилей или динамического управления классами.

Примеры использования:

<div id="parent"> <div id="child"></div> </div> <script> let parent = document.getElementById('parent'); let child = document.getElementById('child'); console.log(parent.contains(child)); // Outputs: true console.log(child.contains(parent)); // Outputs: false </script>

DOMTokenList.contains()

<div id="myElement" class="foo bar"></div> <script> let element = document.getElementById('myElement'); console.log(element.classList.contains('foo')); // Outputs: true console.log(element.classList.contains('baz')); // Outputs: false </script>

Понимание этих методов позволяет вам более эффективно взаимодействовать с DOM в JavaScript и манипулировать им.
