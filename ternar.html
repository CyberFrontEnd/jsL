Все о тернарном операторе

Троичный оператор, часто называемый условным оператором, представляет собой краткий способ написания условных выражений во многих языках программирования. Это называется "троичным", потому что оно принимает три операнда: условие, значение, возвращаемое, если условие истинно, и значение, возвращаемое, если условие ложно. Общий синтаксис троичного оператора таков:

condition ? value_if_true : value_if_false Вот как это работает:

  1. Сначала оценивается условие.

  2. Если условие истинно, выражение возвращает значение, указанное перед : (двоеточие).

  3. Если условие равно false, выражение возвращает значение, указанное после : (двоеточие).

Тернарный оператор часто используется в ситуациях, когда вам нужно присвоить значение переменной на основе условия в одной строке, что делает код более кратким и читаемым.

Однако важно разумно использовать троичный оператор. Если условие сложное или возвращаемые выражения длинные, использование обычного if оператора может быть более читабельным.

Примеры троичного оператора в js 1) Присвоение переменной на основе условия: const age = 25; const allowed = age >= 18 ? "Yes" : "No"; console.log(allowed); // Output: Yes 2) Возвращает значение из функции на основе условия: function getDiscount(amount) { return amount >= 100 ? 0.1 * amount : 0; } console.log(getDiscount(150)); // Output: 15 console.log(getDiscount(50)); // Output: 0 3) Условный рендеринг CSS-класса: const isActive = true; const buttonClass = isActive ? "active" : "inactive"; return <button className={buttonClass}>Click me</button>;
