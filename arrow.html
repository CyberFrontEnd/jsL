Что делает функция () => в js

В JavaScript, выражение () => представляет собой синтаксис стрелочной функции (arrow function). Стрелочные функции представляют более короткий синтаксис для создания функций в JavaScript и были введены в стандарте ECMAScript 6 (ES6).

Основной синтаксис стрелочных функций выглядит так: (parameters) => { // тело функции }

Здесь parameters - это список параметров функции (если они есть), а тело функции заключается в фигурные скобки {}.

Если функция принимает только один параметр, то скобки можно опустить:

parameter => { // тело функции }

Если тело функции состоит из одной инструкции, то фигурные скобки и ключевое слово return можно опустить, и значение этой инструкции буде

parameter => expression

Стрелочные функции имеют некоторые особенности по сравнению с обычными функциями:

  1. Лексическое значение this: Значение this в стрелочной функции принимается из контекста, в котором она была создана, а не из контекста вызова. Это позволяет избежать проблем с this, с которыми сталкиваются в обычных функциях.

  2. Не имеют собственного объекта arguments: В стрелочных функциях нет своего собственного объекта `arguments. Вместо этого можно использовать rest parameters.

  3. Нельзя использовать в качестве конструктора:

    Стрелочные функции не могутnew и вызываnew, они не имеют собственного прототипа и вызовет ошибку, если попытаться использовать их как конструктор.

Вот пример использования стрелочной функции:

const add = (a, b) => { return a + b; } console.log(add(2, 3)); // Выведет: 5 Это function add(a, b) { return a + b; } console.log(add(2, 3)); // Выведет: 5 Стрелочные функции особенно полезны для краткости кода и для использования в контекстах, где важно сохранить значение this.
