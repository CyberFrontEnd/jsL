Конкатенация строк в javascript

В JavaScript вы можете объединять строки с помощью + оператора или с помощью шаблонных литералов (представленных в ES6). Вот как вы можете это сделать.:

  1. Использование оператора +:

    let str1 = "Hello"; let str2 = "world"; let result = str1 + " " + str2; console.log(result); // Output: Hello world

  2. Использование шаблонных литералов:

    let str1 = "Hello"; let str2 = "world"; let result = `${str1} ${str2}`; console.log(result); // Output: Hello world

  3. Литералы шаблонов обеспечивают большую гибкость, поскольку вы можете встраивать выражения или переменные непосредственно в строку, делая ее более читаемой и краткой.

    Вот пример использования выражений внутри литералов шаблона:

    let name = "John"; let age = 30; let result = `My name is ${name} and I am ${age} years old.`; console.log(result); // Output: My name is John and I am 30 years old.

  4. Оба метода достигают одинакового результата, но литералы шаблонов обычно предпочтительнее из-за их удобочитаемости и гибкости.
