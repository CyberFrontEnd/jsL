что делает функция addEventListener в js

Функция addEventListener() в JavaScript используется для добавления обработчика событий к определенному элементу DOM. Этот метод позволяет привязать функцию (обработчик) к определенному событию (например, клик, наведение мыши, нажатие клавиши и т. д.), которое может произойти с этим элементом.

element.addEventListener(event, handlerFunction);

Вот что представляет каждый параметр:

  • element: Элемент DOM, к которому вы хотите добавить обработчик событий.
  • event Строка, указывающая тип события, к которому вы хотите привязать обработчик.
  • handlerFunction Функция, которая будет вызвана при наступлении события. Эта функция может принимать аргумент события (обычно обозначается как event), который содержит информацию о событии.

Пример использования addEventListener():

const button = document.getElementById('myButton'); function handleClick(event) { console.log('Кнопка была нажата'); } button.addEventListener('click', handleClick);

В этом примере при нажатии на кнопку с id "myButton" будет вызвана функция handleClick(), которая выводит сообщение в консоль

Одним из преимуществ `adaddEventListener() является то, что вы можете добавлять несколько обработчиков для одного события на один и тот же элемент. Это делает его гибким и удобным инструментом для работы с событиями в JavaScript. Кроме того, с помощью removeEventListener() вы можете удалить обработчики событий, если они больше не нужны, что помогает избежать утечек памяти и других проблем.
