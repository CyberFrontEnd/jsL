Что такое функция выборки в javascript?

Функция fetch() в JavaScript используется для выполнения сетевых запросов для извлечения ресурсов (таких, как данные JSON, изображения, файлы и т.д.) С сервера или другого ресурса в сети. Это современная замена старым API, таким как XMLHttpRequest.

Вот простой пример того, как вы могли бы использовать fetch() для выполнения запроса GET для извлечения данных JSON:

fetch('https://api.example.com/data.json') .then(response => { if (!response.ok) { throw new Error('Network response was not ok'); } return response.json(); }) .then(data => { console.log(data); // Do something with the JSON data }) .catch(error => { console.error('There was a problem with the fetch operation:', error); });

В этом примере:

  1. fetch('https://api.example.com/data.json') инициирует GET-запрос на указанный URL-адрес.
  2. .then(response => { ... }) обрабатывает ответ от сервера.
  3. Внутри первого .then() блока мы проверяем, прошел ли ответ успешно, используя response.ok. Если это не так, мы выдаем ошибку.
  4. Если с ответом все в порядке, мы анализируем тело ответа как JSON с помощью response.json() , который возвращает обещание, содержащее проанализированные данные JSON.
  5. Затем у нас есть еще один .then() блок, где мы можем использовать данные JSON.
  6. Если во время операции выборки возникают какие-либо ошибки, они фиксируются в .catch() блоке.
