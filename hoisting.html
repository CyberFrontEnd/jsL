Понятие "подъем" (hoisting)

Понятие "подъем" (hoisting) в JavaScript относится к тому, как интерпретатор JavaScript обрабатывает объявления переменных и функций. Давайте разберем это подробнее.

В JavaScript объявления переменных (с ключевыми словами var, let, и const) поднимаются (hoisted) наверх их области видимости. Однако, инициализация переменных остается на месте. Это значит, что переменная может быть использована до того, как она была объявлена, но до инициализации её значение будет undefined (для var). Для let и const переменных, доступ к переменной до объявления приведет к ошибке ReferenceError.

console.log(a); // undefined var a = 5; console.log(a); // 5

Это эквивалентно:

var a; console.log(a); // undefined a = 5; console.log(a); // 5

Пример с let и const:

console.log(b); // ReferenceError: Cannot access 'b' before initialization let b = 5; console.log(b); // 5 console.log(c); // ReferenceError: Cannot access 'c' before initialization const c = 10; console.log(c); // 10

Подъем функций

Функции, объявленные с помощью function declaration, поднимаются целиком, включая их тело. Это значит, что функция может быть вызвана до её объявления в коде.

console.log(foo()); // "Hello" function foo() { return "Hello"; }

Это эквивалентно:

function foo() { return "Hello"; } console.log(foo()); // "Hello"

Однако, это не относится к функциям, объявленным с помощью function expression. В этом случае поднимается только объявление переменной, а не самой функции.

console.log(bar()); // TypeError: bar is not a function var bar = function() { return "Hi"; }; console.log(bar()); // "Hi"

Это эквивалентно:

var bar; console.log(bar()); // TypeError: bar is not a function bar = function() { return "Hi"; }; console.log(bar()); // "Hi"

Заключение

Подъем (hoisting) в JavaScript - это механизм, с помощью которого объявления переменных и функций "поднимаются" к началу их области видимости. Важно понимать этот механизм, чтобы избежать неожиданных результатов при доступе к переменным и функциям до их явного объявления в коде.
