что делает метод reduce в js

Метод reduce() в JavaScript используется для сведения массива к одному значению. Он выполняет предоставленную функцию обратного вызова для каждого элемента массива, в результате чего получается одно выходное значение. Этот метод выполняет итерацию по массиву слева направо, накапливая единственный результат путем применения функции обратного вызова к каждому элементу.

Вот синтаксис метода reduce():
array.reduce(callback(accumulator, currentValue, currentIndex, array), initialValue)

  1. callback: Функция, выполняемая для каждого элемента массива. Принимает четыре аргумента:

  2. accumulator: Накопленное значение, ранее возвращенное обратным вызовом, или initialValue, если указано. Он накапливает возвращаемые обратным вызовом значения.

  3. currentValue: Текущий элемент, обрабатываемый в массиве.

  4. currentIndex (необязательно): индекс текущего обрабатываемого элемента в массиве.

  5. array (необязательно): Был вызван массив reduce().

initialValue (необязательно): значение для использования в качестве начального значения для накопителя. Если не указано, в качестве начального значения накопителя будет использоваться первый элемент массива, и итерация начнется со второго элемента.

Метод reduce() выполняет функцию обратного вызова один раз для каждого элемента, присутствующего в массиве, за исключением первого элемента, если он предоставлен initialValue. Он принимает возвращаемое значение обратного вызова и использует его в качестве накопительного значения для следующей итерации. Наконец, он возвращает накопленный результат после перебора всего массива.

const numbers = [1, 2, 3, 4, 5]; const sum = numbers.reduce((accumulator, currentValue) => accumulator + currentValue, 0); console.log(sum); // Output: 15 (1 + 2 + 3 + 4 + 5)

accumulator Начинается с начального значения 0.

Функция обратного вызова добавляет каждый из них currentValue в accumulator.

После перебора всех элементов массива в качестве выходных данных возвращается конечное накопленное значение (15).

примеры метода reduce() в js

  1. Суммирование элементов массива:

    const numbers = [1, 2, 3, 4, 5]; const sum = numbers.reduce((accumulator, currentValue) => accumulator + currentValue, 0); console.log(sum); // Output: 15 (1 + 2 + 3 + 4 + 5)

  2. Вычисление произведения элементов массива:

    const numbers = [1, 2, 3, 4, 5]; const product = numbers.reduce((accumulator, currentValue) => accumulator * currentValue, 1); console.log(product); // Output: 120 (1 * 2 * 3 * 4 * 5)

  3. Выравнивание массива массивов:

    const nestedArray = [[1, 2], [3, 4], [5, 6]]; const flattenedArray = nestedArray.reduce((accumulator, currentValue) => accumulator.concat(currentValue), []); console.log(flattenedArray); // Output: [1, 2, 3, 4, 5, 6]

  4. Подсчет вхождений элементов в массив:

    const colors = ['red', 'blue', 'red', 'green', 'blue', 'red']; const colorCount = colors.reduce((accumulator, currentValue) => { accumulator[currentValue] = (accumulator[currentValue] || 0) + 1; return accumulator; }, {}); console.log(colorCount); // Output: { red: 3, blue: 2, green: 1 }

  5. Группировка объектов по свойству:

    const people = [ { name: 'Alice', age: 30 }, { name: 'Bob', age: 25 }, { name: 'Charlie', age: 35 }, { name: 'David', age: 30 } ]; const groupedByAge = people.reduce((accumulator, currentValue) => { const key = currentValue.age; if (!accumulator[key]) { accumulator[key] = []; } accumulator[key].push(currentValue); return accumulator; }, {}); console.log(groupedByAge); /* Output: { 30: [ { name: 'Alice', age: 30 }, { name: 'David', age: 30 } ], 25: [ { name: 'Bob', age: 25 } ], 35: [ { name: 'Charlie', age: 35 } ] } */

Эти примеры демонстрируют различные варианты использования reduce() метода в JavaScript, от простых арифметических операций до более сложных задач, таких как выравнивание массивов или группировка объектов.
