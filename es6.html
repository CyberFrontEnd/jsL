Примеры ES6

  1. Функции со стрелками:

    // ES5 var multiply = function(x, y) { return x * y; }; // ES6 const multiply = (x, y) => x * y;

  2. Литералы шаблона:(Конкатенация)

    // ES5 var name = 'John'; console.log('Hello, ' + name + '!'); // ES6 const name = 'John'; console.log(`Hello, ${name}!`);

  3. let и Const:

    // ES5 var x = 10; x = 20; // ES6 let x = 10; const y = 20;

  4. Классы:

    // ES5 function Person(name) { this.name = name; } Person.prototype.sayHello = function() { console.log('Hello, ' + this.name); }; // ES6 class Person { constructor(name) { this.name = name; } sayHello() { console.log(`Hello, ${this.name}`); } }

  5. Назначение Де структурирования:

    // ES5 var person = { name: 'John', age: 30 }; var name = person.name; var age = person.age; // ES6 const person = { name: 'John', age: 30 }; const { name, age } = person;

  6. Оператор распространения:

    // ES5 var arr1 = [1, 2, 3]; var arr2 = [4, 5, 6]; var arr3 = arr1.concat(arr2); // ES6 const arr1 = [1, 2, 3]; const arr2 = [4, 5, 6]; const arr3 = [...arr1, ...arr2];
