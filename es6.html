Примеры ES6

  1. Функции со стрелками:

    // ES5 var multiply = function(x, y) { return x * y; }; // ES6 const multiply = (x, y) => x * y;

  2. Литералы шаблона:(Конкатенация)

    Литералы шаблонов упрощают интерполяцию строк и многострочных строк.

    // ES5 var name = 'John'; console.log('Hello, ' + name + '!'); // ES6 const name = 'John'; console.log(`Hello, ${name}!`);

  3. let и Const:

    Объявления Let и Const: let и const предоставляют переменные с блочной областью, заменяя var.

    // ES5 var x = 10; x = 20; // ES6 let x = 10; const y = 20;

  4. Классы:

    Введен синтаксис классов для упрощения объектно-ориентированного программирования на основе прототипов.

    // ES5 function Person(name) { this.name = name; } Person.prototype.sayHello = function() { console.log('Hello, ' + this.name); }; // ES6 class Animal { constructor(name) { this.name = name; } speak() { console.log(`${this.name} makes a noise.`); } } const dog = new Animal('Dog'); dog.speak(); // Outputs: Dog makes a noise.

  5. Назначение Де структурирования:

    Легко извлекайте значения из массивов или объектов и присваивайте их переменным.

    // ES5 var person = { name: 'John', age: 30 }; var name = person.name; var age = person.age; // ES6 const person = { name: 'John', age: 30 }; const { name, age } = person; // Outputs: John 30 const person = { name: "Alice", age: 30 }; const { name, age } = person; console.log(name, age); // Outputs: Alice 30

  6. Оператор распространения:

    Позволяет расширять итеративное значение, такое как выражение массива или строка, в местах, где ожидается ноль или более аргументов.

    // ES5 var arr1 = [1, 2, 3]; var arr2 = [4, 5, 6]; var arr3 = arr1.concat(arr2); // ES6 const arr1 = [1, 2, 3]; const arr2 = [4, 5, 6]; const combined = [...arr1, ...arr2]; console.log(combined); // Outputs: [1, 2, 3, 4, 5, 6]

  7. Обещания

    Упрощает асинхронное программирование, упрощая работу с асинхронными операциями.

    function fetchData() { return new Promise((resolve, reject) => { // Asynchronous operation setTimeout(() => { resolve('Data fetched successfully!'); }, 2000); }); } fetchData() .then(data => console.log(data)) .catch(error => console.error(error));

Это некоторые из основных функций, представленных в ES6, каждая из которых предоставляет новые мощные способы написания кода JavaScript.
