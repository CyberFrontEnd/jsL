что делает функция forEach в js

В JavaScript forEach() функция - это метод, доступный в массивах, который позволяет вам выполнять итерации по каждому элементу в массиве и выполнять предоставленную функцию для каждого элемента. Цель этой функции - выполнить определенное действие для каждого элемента в массиве без необходимости явной итерации с использованием for цикла.

Основной синтаксис для forEach() является:

array.forEach(function(element, index, array) { // Do something with each element });

Вот что представляет каждый параметр:

  • element: Текущий элемент, обрабатываемый в массиве.
  • index (необязательно): индекс текущего обрабатываемого элемента.
  • array (необязательно): был вызван массив forEach().

Функция, переданная forEach() в качестве аргумента, будет выполняться один раз для каждого элемента в массиве, и вы можете выполнить любую желаемую операцию внутри этой функции.

let numbers = [1, 2, 3, 4, 5]; numbers.forEach(function(number) { console.log(number); }); Результат

1 2 3 4 5
