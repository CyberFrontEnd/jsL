что делает textContent в javascript

В JavaScript свойство textContent используется для получения или установки текстового содержимого узла и всех его потомков. Оно возвращает текстовое содержимое в виде строки. Вот как это работает:

Получение текстового содержимого:

const element = document.getElementById('example'); const text = element.textContent; console.log(text); // Outputs the text content of the element

Настройка текстового содержимого:

const element = document.getElementById('example'); element.textContent = 'New text content';

Пример:

This is some bold text.

const element = document.getElementById('example'); console.log(element.textContent); // Outputs: "This is some bold text."

В этом примере element.textContent возвращает объединенное текстовое содержимое и всех его потомков, включая элементы и.

В отличие от innerHTML, textContent возвращает только текстовое содержимое без какой-либо HTML-разметки. Часто используется, когда вам нужно работать только с самим текстом и вы не хотите иметь дело с HTML-тегами. Кроме того, использование textContent может помочь предотвратить уязвимости XSS (межсайтового скриптинга), поскольку оно обрабатывает содержимое как обычный текст, а не как HTML.
